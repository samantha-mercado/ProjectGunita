SHARING IS CARING!
Project Gunita values the accessibility of all educational material in order for every child to have an easier and more fun learning experience. With your help, we can create a growing library of free educational resources for young learners, teachers, and parents all across the Philippines.
TEACHER'S CORNER FAQS
  • Anyone can upload Word documents, PDFs, pictures, videos, presentations, worksheets. any file type of any size. Make sure that all files have appropriate file names before uploading.

    Once they have been uploaded, the Project Gunita team will review the content (just to double-check if the files are safe and genuine!) after which the materials will be ready for viewing and download for other users of the website. Salamat!
  • The Project Gunita team will gladly layout any material submitted if it is text-heavy or in need of a design update!

    By submitting your educational resources, you grant Project Gunita permission to modify and distribute any submitted material and acknowledge that all submitted resources will be available for download by other users of the Project Gunita website.
